# Work Interruptions and Childcare: Exploring the Child Penalty in the Labour Market

**David Felipe-Calvo · José Ignacio Giménez-Nadal · José Alberto Molina**  
University of Zaragoza · IEDIS

---

## Abstract

Using harmonised time-use data from the Multinational Time Use Study (MTUS, 2007–2023), we analyse the daily risk that working parents interrupt paid work to provide childcare. Cox proportional-hazards models are estimated with survey weights and household-clustered standard errors; SHAP values and Accumulated Local Effects plots characterise predictive heterogeneity. The results reveal that the most powerful predictor of interruption risk is parental gender (HR = 0.388; fathers 61% less likely to interrupt than mothers), followed by the age of the youngest child and family status, surpassing economic variables such as income. A cross-national extension to eight countries (n = 42,344; 1,915 events) confirms that the gender gap is universal: male hazard ratios range from 0.337 (United States) to 0.655 (United Kingdom).

## Data

The analysis uses the [Multinational Time Use Study (MTUS)](https://www.timeuse.org/mtus), harmonised release. Data are not included in this repository due to redistribution restrictions. Researchers can request access directly from the MTUS project.

Processed `.RData` files are generated by running `01_ext_muestra.R` followed by `02_ext_filtrado.R` against the raw MTUS CSV files.

## Reproducing the results

1. Obtain the MTUS harmonised files (`MTUS_haf.csv`, `MTUS_hef1.csv`, `MTUS_hef2.csv`) and place them in `data/`.
2. Run `code/01_ext_muestra.R` to extract the US and multi-country samples.
3. Run `code/02_ext_filtrado.R` to build the cleaned analysis datasets.
4. Run `code/gen_fig1.R` to generate Figure 1.
5. Run `code/run_analisis_v2.R` to reproduce all models, figures, and the analysis log.

Figures are saved to `output/figures/`; the analysis log to `output/logs/output_log_v2.txt`.
